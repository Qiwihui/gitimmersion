沉浸式学 Git
==================

讲解 Git 的基本原理。要了解一件事情最好的方法就是去做。

Git 是一个功能强大，用于分布式版本控制的复杂系统。对功能的理解为开发人员提供了一种开放的源代码管理的新方法。
掌握 Git 的最可靠途径是沉浸在于实用程序和操作中，亲身体验。

- `安装Git <http://git-scm.com/download>`_
- `图形客户端 <https://git-scm.com/downloads/guis>`_
- `Github Issues <https://github.com/qiwihui/git_immersioncn/issues>`_
